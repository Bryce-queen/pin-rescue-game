# 拉针救援 - 原创微信小游戏 MVP

一个“广告同款但真实可玩”的原创拉针解谜小游戏：玩家按顺序拔针，让水灭火、避开怪物、救到人质并拿到宝箱。

## 玩法

- 点击金色拉针即可拔出。
- 拔针后，水、火、怪物、宝箱会沿开放路径下落或移动。
- 水遇火会灭火。
- 怪物遇火会被消灭，怪物碰到人质会失败。
- 宝箱或人质到达出口即满足目标。
- 每关目标可能是救人、拿宝箱、灭火或消灭怪物。

## 运行

1. 打开微信开发者工具。
2. 选择“小游戏”。
3. 导入本目录：`pin-rescue-game`。
4. AppID 可先选择测试号或游客模式。

## 当前版本

- 原生 Canvas，无外部素材依赖。
- 内置 20 个原创关卡。
- 简化格子物理，适合 MVP 快速验证。
- 启动页、关卡结算、分享入口、失败重试。
